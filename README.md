# Contract

Node: v18.12.1
Npm: 8.19.2

## Usage

### 1. Build contract

```
npm run spec:build
```

### 2. Build DTO's

```
npm run spec:serve
npm run dtos:build
```

### 3. Run mock

```
npm run mock:server
npm run mock:proxy
npm run mock:static
npm run mock:eventsource
npm run mock:websocket
```

### 4. Run contract preview

```
npm run spec:preview
```
